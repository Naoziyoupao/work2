# 项目说明（分类器训练精简版）

## 1. 迁移目标

本项目从原工程迁移而来，目标是保留最小可用的“训练分类器”流程，删除与训练主流程无关的实验与分析内容。

## 2. 保留内容

- 分类器训练脚本：
  - `scripts/train/train_classifier_new.py`
  - `scripts/train/train_classifier.py`
- OASST2 数据处理脚本：
  - `scripts/data/process_oasst2.py`
  - `scripts/data/prepare_classification_data.py`
  - `scripts/data/prepare_classification_data_response_only.py`
- OASST2 数据目录结构：
  - `data/raw/oasst2/`
  - `data/processed/oasst2/`

## 3. 删除内容

以下内容不再保留：

- PCA / 困惑度 / 多层特征分析脚本
- 其他实验（exp002+）相关文档与辅助脚本
- 与分类器训练无关的数据集与流程

## 4. 运行建议

1. 安装依赖：`pip install -r requirements.txt`
2. 需要时重新处理 OASST2 数据
3. 使用 `train_classifier_new.py` 作为主训练入口

## 5. Git 管理

- 已在新目录初始化独立 Git 仓库
- 使用 `.gitignore` 忽略模型权重、缓存、日志及大文件
